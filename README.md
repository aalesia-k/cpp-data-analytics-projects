# C++ Data Analytics Projects

This repository contains C++ programs focused on data analysis, statistics, and structured data processing.  
Developed as part of coursework at California State University San Marcos.

## Projects

### 1. K-Pop Streaming Analytics System
A menu-driven application that:
- Loads artist data from a file
- Calculates average monthly streams
- Sorts artists alphabetically and by listener count
- Searches and removes artists
- Performs cross-artist statistical comparisons

Concepts Used:
- Structs
- File I/O
- Arrays
- Sorting algorithms
- Modular programming

---

### 2. California Fuel Price Analysis
Analyzes fuel price data across counties and compares:
- Current averages
- Month-old averages
- Year-old averages
- Lowest price detection
- Threshold-based filtering

Concepts Used:
- Arrays
- Aggregation functions
- Index tracking
- Formatted output

---
