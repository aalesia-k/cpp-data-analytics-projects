# K-Pop Streaming Analytics System

## Overview
A C++ console application that processes Spotify streaming data for multiple artists across five months.

## Features
- Load artist data from file
- Search artist by name
- Calculate average monthly streams
- Identify highest listener count by month
- Sort artists alphabetically
- Sort artists by average listeners
- Remove an artist dynamically

## Technical Implementation
- Struct-based data modeling
- File input handling
- Menu-driven interface
- Custom sorting algorithms
- Statistical aggregation functions

## How to Run
Compile using:

g++ main.cpp -o program
./program
